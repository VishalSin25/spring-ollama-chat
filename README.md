# Spring AI Chat with Ollama

A simple **Spring Boot** application integrated with **Ollama AI** to provide chatbot responses, with a modern static frontend.

---

## Features

- REST API with Spring Boot
- Chat with Ollama AI using `/api/ollama/{message}`
- Modern chat-style frontend with colored chat bubbles
- Simple, lightweight frontend served directly by Spring Boot
- Easily extendable to React or other frontend frameworks
